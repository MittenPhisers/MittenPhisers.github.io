De Fysieke Laag

Inleiding

De fysieke laag van een apparaat is alle hardware. Dit zijn alle draden, de accu, het geheugen, de processor, het motherboard, het touchscreen etc. Alle onderdelen waar de telefoon uit bestaat dus. Deze pagina legt alle belangrijke onderdelen uit.

Motherboard

Het Motherboard wordt vaak gezien als het 'hart' van een computer. Het Motherboard bestaat uit de belangrijkste onderdelen van een apparaat, om ze zo op een centraal punt te verbinden. Hier op is bijvoorbeeld het CPU gemonteerd, het brein van de computer. Of de RAM, het geheugen van de computer. Op het Motherboard zitten ook stroomdraden, USB poorten en circuits om de hardware met elkaar te verbinden.

RAM

De RAM is een snelle geueugen cache die ervoor zorgt dat de CPU snel data op kan slaan. Dit is nodig zodat het brein van de computer gegevens kan vinden, zelfs als het ergens anders opgeslagen is, zonder de RAM zou het systeem niet efficiÃ«nt werken. In de RAM wordt informatie tijdelijk opgeslagen, als de CPU bijvoorbeeld iets aan het bewerken is, dan wordt dit opgeslagen in de RAM, zodat het hier makkelijk bij kan. RAM verschilt dus hiermee van de harde schijf, waar informatie permanent opgeslagen wordt.

Processor

De Processor, ook wel 'CPU' genoemd, is het belangrijkste component van een computer; het voert namelijk de code uit. Elke keer dat je telefoon iets doet, dan voert de processor deze taak uit. De processor is verantwoordelijk voor berekeningen, gegevensbewerking en het uitvoeren van programma's, een videogame bijvoorbeeld. Moderne processors zijn razend snel en kunnen duizenden gegevens verwerken in slechts een aantal seconden.
