    De Toepassingslaag

    Inleiding

    De toepassingslaag is de toepassing van alle componenten in een apparaat. Van een smartphone zijn dit de apps, van een tv zijn dit de programmas die je kunt kijken, van een radio is dit de muziek die je kunt horen; alles wat een apparaat doet als functie, is de toepassingslaag.

    Toepassingsprogramma's

    Toepassingsprogramma's, ook wel bekend als softwaretoepassingen of gewoon 'apps', zijn computerprogramma's die zijn ontworpen om specifieke taken of functies uit te voeren voor gebruikers. Deze programma's worden gebruikt om verschillende soorten taken en activiteiten uit te voeren op computers, smartphones, tablets en andere apparaten. Als je een app download, download je dus eigenlijk een script die uitgevoerd wordt door de processor. Deze scripten zijn specifiek ontworpen voor gebruikers.

    API

    Een API, afgekort voor Application Programming Interface, is een set regels en protocollen waarmee verschillende software kan communiceren en samenwerken met elkaar. Een API definieert de manier waarop verschillende stukken software met elkaar kunnen praten en gegevens kunnen uitwisselen, zonder dat ontwikkelaars de interne werking van die software volledig hoeven te begrijpen. Zonder een API zou software zwaar chaotisch zijn, doordat ontwikkelaars per toepassingsprogramma apart samenwerking moeten programmeren.
